# 贡献指南

感谢你对 TokenSense 的兴趣！欢迎贡献代码。

## 快速开始

1. Fork 这个仓库
2. 克隆你的 Fork：`git clone https://github.com/YOUR_USERNAME/token-dashboard.git`
3. 创建功能分支：`git checkout -b feature/your-feature`

## 开发环境

```bash
cd src
python3 refresh_token_data.py --once
open token_visual.html
```

## 代码规范

- Python：遵循 PEP 8，使用 flake8 检查
- JavaScript：使用 ES6+ 语法
- CSS：遵循现有样式约定

## 提交 Pull Request

1. 推送你的分支：`git push origin feature/your-feature`
2. 在 GitHub 上创建 Pull Request
3. 填写 PR 模板中的相关信息

## 问题反馈

- Bug 报告：使用 [Issue Template](.github/ISSUE_TEMPLATE/bug_report.md)
- 功能建议：使用 [Feature Request Template](.github/ISSUE_TEMPLATE/feature_request.md)

感谢你的贡献！
