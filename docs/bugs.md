# TokenSense Bug 跟踪

## 问题列表

### 问题 3：热力图图例缺少悬浮提示
- **位置**：Tokens 标签页 → 活动热力图 → 底部图例（少~多之间的小方块）
- **现象**：鼠标悬浮在图例小方块上没有 tooltip 显示代表的 token 数值范围
- **优先级**：低

---

### 问题 4：Costs 模块多个指标无数据/格式异常
- **位置**：Costs 标签页
- **现象**：
  - "实际记录" 和 "最高单次" 显示未格式化浮点数（如 $5578.294854300007）
  - "平均单轮" 显示 "--分钟"
  - "自动化率" 显示 "--"
  - "AI 计算时间" 显示 "0小时"，副标题 "平均单轮 --分钟"
  - "自动化比例" 显示 "--"，副标题 "0 次会话超过 90%"
- **根因**：`turn_duration_stats` 和 `agentic_stats` 在 server.js 中未计算，返回空对象 `{}`
- **优先级**：高

---

### 问题 5：响应时间分析图表无数据
- **位置**：Costs 标签页 → "响应时间" 图表区域
- **现象**：显示"无单轮耗时数据"，图表完全空白
- **根因**：`turn_duration_stats` 未计算（与问题 1、4 同根因）
- **优先级**：高

---

### 问题 9：自主性趋势图无数据
- **位置**：Behavior 标签页 → "自主性趋势" 图表区域
- **现象**：整个图表区域完全空白，无数据展示
- **根因**：`autonomy_trend` 在 server.js 中返回空数组 `[]`，未从 session 权限模式数据中计算
- **优先级**：中

---

### 问题 8：工作节奏条形图不显眼
- **位置**：Behavior 标签页 → 侧栏"工作节奏" → 周一~周日条形图
- **现象**：条形图填充颜色与背景色太接近，几乎看不出来，右侧数字堆叠重叠
- **待确认**：CSS 中 `.h-bar-fill` 缺少具体颜色定义（只有 cost/mcp/skill/time 等子类，weekday 图用的是 `cool` 类但未定义对应颜色）
- **优先级**：中

---

### 问题 7：活跃天数/连续天数统计偏低
- **位置**：Behavior 标签页 → 顶部"使用行为" → 活跃天数、连续天数
- **现象**：显示活跃 31 天、连续 30 天、最高 30 天。用户反馈最近每天都有活跃，数值应该更高
- **待确认**：活跃天数统计的日期对齐逻辑（`s.date` 字段使用本地时间 `toLocaleDateString('en-CA')`，可能与会话实际创建日期有偏差）
- **优先级**：中

---

### 问题 6：Stability 模块全部为 0
- **位置**：Costs 标签页 → Stability 卡片
- **现象**：API Errors、Max Retries、Context Compacts 全部显示 0，图表为空
- **根因**：`stability` 在 server.js 中硬编码返回 `{ total_api_errors: 0, total_compact_events: 0, sessions_with_errors: 0 }`，未从 session 数据中计算
- **优先级**：中

---

### 问题 5：AI 计算时间和平均响应时间图表无数据
- **位置**：Costs 标签页 → 响应时间分析图表
- **现象**：图表区域完全空白，无数据展示
- **根因**：同上，`turn_duration_stats` 未计算
- **优先级**：高

---

### 问题 2：小时级图表 X 轴不显示
- **位置**：Tokens 标签页 → 每日用量图表 → 切换到"按时"模式
- **现象**：图表有数据（折线和点都显示），但底部 X 轴没有小时标签（红框区域为空）
- **根因**：待查，可能是 Chart.js X 轴刻度配置问题
- **优先级**：中

---

### 问题 1：AI 计算时间卡片不显示数据
- **位置**：Tokens 标签页 → 顶部指标卡片 → "AI 计算时间"
- **现象**：只显示 "—"，没有数值和副标题
- **根因**：`turn_duration_stats` 在 server.js 中未计算，返回空对象 `{}`。`renderHero` 中 `s.turn_duration_stats.total_hours` 为 undefined
- **优先级**：中

---

### 问题 10：工具模块两个问题
- **位置**：Tools 标签页
- **子问题 10a：工具链无数据**

  - 优先级：中
- **子问题 10b：工具排行榜条形图太短太粗，需要重新设计**
  - 现象：条形图过短，视觉不够美观，右侧百分比全部 100% 意义不大
  - 优先级：中
